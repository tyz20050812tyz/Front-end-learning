## 美团网项目 🛍️

这是一个完整的美团网移动端页面项目，基于实际的美团网界面进行开发，包含了完整的首页展示、商品列表、搜索功能等核心功能。

### 项目结构
```
美团网项目/
├── css/                    # 样式文件
│   ├── iconfont.css       # 图标字体样式
│   ├── iconfont2.css      # 图标字体样式2
│   └── index.css          # 主样式文件
├── font/                  # 字体文件
│   ├── iconfont.woff      # 图标字体文件
│   ├── iconfont2.eot      # 图标字体文件
│   ├── iconfont2.svg      # 图标字体文件
│   ├── iconfont2.ttf      # 图标字体文件
│   └── iconfont2.woff2    # 图标字体文件
├── img/                   # 图片资源
│   ├── banner.jpg         # 横幅图片
│   ├── download_banner.png # 下载横幅
│   ├── food.jpg           # 美食图片
│   ├── ktv.jpg            # KTV图片
│   └── nav_1.png          # 导航图片
└── index.html             # 主页面文件
```

### 功能特性 ✨
- 🏠 **完整首页**: 包含城市选择、搜索框、轮播图、导航菜单
- 🛍️ **商品展示**: 拼团商品、猜你喜欢等商品列表展示
- 📱 **移动端适配**: 响应式设计，完美适配手机屏幕
- 🎨 **精美样式**: 采用美团经典配色方案，视觉效果优秀
- 🔍 **搜索功能**: 顶部搜索框，支持商品和店铺搜索
- 📊 **商品信息**: 展示商品图片、名称、价格、销量等完整信息

### 技术亮点 🚀
- **Flex布局**: 大量使用Flex布局实现响应式设计
- **CSS3特性**: 使用圆角、阴影、渐变等CSS3特性
- **字体图标**: 使用iconfont字体图标，减少图片资源
- **移动端优化**: 针对触摸操作进行优化
- **性能优化**: 合理的文件结构和样式组织

### 页面结构 📱
1. **头部区域**: 城市选择 + 搜索框
2. **轮播图**: 广告banner展示
3. **导航菜单**: 8个功能入口图标
4. **拼团区域**: 热门拼团商品展示
5. **猜你喜欢**: 个性化商品推荐

### 学习价值 📚
这个项目将帮助你：
- 🎯 掌握移动端页面开发技巧
- 🎨 学习实际项目的CSS架构设计
- 📐 理解响应式布局的实现方法
- 🛠️ 熟悉商业级项目的开发流程
- 🎭 提升页面美化和用户体验设计能力

## 使用说明 💡

这个项目主要用于学习和练习HTML和CSS基础知识，可以直接在浏览器中打开HTML文件查看效果。

### 快速开始

```bash
# 克隆项目
git clone <项目地址>

# 进入项目目录
cd 前端学习

# 直接在浏览器中打开HTML文件进行学习
```

### 美团网项目运行

```bash
# 进入美团项目目录
cd 美团网项目

# 使用Live Server或其他工具打开index.html
# 或者在浏览器中直接打开index.html文件
```

### 学习建议

1. **按章节顺序学习**: 建议按照章节顺序逐步学习，先掌握HTML基础再学习CSS样式设计。
2. **动手实践**: 每个章节都包含示例代码，建议在学习过程中动手实践，加深理解。
3. **完成练习题**: 每章末尾的练习题可以帮助检验学习效果，建议认真完成。
4. **查阅文档**: 遇到不理解的概念时，可以查阅相关文档或在线资源。
5. **项目实战**: 完成基础学习后，可以尝试修改美团项目，加深理解
